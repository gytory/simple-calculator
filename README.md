# simple-calculator

Simple calculator - React приложение, выполняющее базовый функционал калькулятора. 

## Функционал

- Сложение двух чисел
- Вычитание одного числа из другого
- Умножение чисел
- Деление чисел с защитой от деления на ноль
- Очистка вычислений
- Поддержка десятичных чисел
- Обработка ошибок (деления на ноль)

## Как пользоваться

1. Ввод чисел: нажимайте цифровые кнопки от 0 до 9
2. Операции: используйте кнопки +, -, *, / для выбора математической операции
3. Результат: нажмите = для выполнения вычисления
4. Очистка: нажмите C для сброса калькулятора
5. Десятичные числа: используйте . для ввода дробных чисел
